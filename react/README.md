# react-camp-boilerplate

## Folder structure

```
src
├── api
│   ├── dtos
│   ├── mappers
│   └── services
├── assets
├── components
├── features
├── models
├── routes
│   └── guards
├── store
├── theme
└── utils
```

## Component file structure

```
CustomButton
├── CustomButton.tsx
├── CustomButton.module.css
└── index.ts
```

## Install dependencies

```bash
npm run install
```

## Development mode

```bash
npm run start
```

## Lint

```bash
npm run lint
```

## Lint with fix

```bash
npm run lint:fix
```
